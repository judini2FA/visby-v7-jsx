# Extension Icons

Drop three PNG files here before loading the extension:

| File        | Size    |
|-------------|---------|
| icon16.png  | 16x16   |
| icon48.png  | 48x48   |
| icon128.png | 128x128 |

The extension loads fine without them (Chrome will show a warning, not an error). Missing icons won't block `chrome://extensions → Load unpacked`.

Suggested content: the Visby hexagon logo mark in white on a transparent background, or on the brand gradient `linear-gradient(135deg,#25CDB8,#2A8AED 50%,#BC2DE6)`.
